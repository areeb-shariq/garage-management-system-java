# Garage Management System

## Overview

Garage Management System is a Java-based console application developed as part of an Object-Oriented Programming project.

The system allows garage staff to register customers, store vehicle information, select service packages, choose individual services, and generate bills with discounts.

Customer records are stored using file handling, allowing existing customers to be retrieved using their customer ID.

## Features

- New customer registration
- Existing customer lookup
- Vehicle information management
- Service package selection
- Individual service selection
- Automatic bill generation
- Discount calculation
- File-based customer record storage

## OOP Concepts Used

- Abstraction
- Inheritance
- Encapsulation
- Polymorphism

## Technologies Used

- Java
- Object-Oriented Programming
- File Handling
- Collections Framework
